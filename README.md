# Unlock

A decentralized content platform with client-side AES encryption, IPFS storage, x402 micropayments, and on-chain access control.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Next.js 14** - React framework with App Router
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - UI component library
- **RainbowKit** - Wallet connection
- **wagmi/viem** - Ethereum interactions
- **Polygon Amoy** - Testnet deployment
- **Pinata IPFS** - Decentralized storage
- **x402** - HTTP micropayments

## Features

- Client-side AES-GCM encryption
- IPFS content storage via Pinata
- x402 micropayments for content access
- On-chain access registry
- Creator dashboard with earnings tracking
