## Content Guidelines

1. **Keep files small** - Under 10MB for quick uploads during demo
2. **Use descriptive titles** - Makes browsing more realistic
3. **Vary prices** - Shows flexibility of the platform
4. **Cover multiple categories** - Demonstrates filtering feature
